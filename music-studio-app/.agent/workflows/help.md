# GSD Commands

- **/new-project**: Initialize a new project (Spec & Roadmap).
- **/map**: Analyze codebase and update Architecture.
- **/plan**: Create a technical plan for a phase.
- **/execute**: Execute the tasks in the current plan.
- **/verify**: Verify the implemented tasks.
- **/progress**: Show current project status.
- **/help**: Show this help message.

**Workflow**:

1. `/new-project` -> 2. `/map` -> 3. `/plan` -> 4. `/execute` -> 5. `/verify` -> Repeat from `/plan`.
